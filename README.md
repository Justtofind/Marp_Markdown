# markdown_ppt
将markdown直接转为ppt slides

## 操作步骤
### 1. 在vscode中下载安装Marp for vscode 拓展
### 2. 使用已经准备好的主题
* 下载themes和settings.json文件到对应markdown笔记的文件夹
### 3. 使用方法
* 在markdown文件最开头加入如下代码即可开始使用
```markdown
---
marp: true
size: 16:9  
theme: am_blue 
paginate: true
headingDivider: [2,3]
footer: nothy 
---
```
* 熟悉marp基本语法
`---`表示分页
